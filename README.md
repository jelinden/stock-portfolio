# Run

`npm run build && go build && ./stock-portfolio`

# Simple load test

`go test`

# Development stack

Fontend: Reactjs

Backend: Golang, http router github.com/julienschmidt/httprouter

Database: Embedded github.com/cznic/ql / modernc.org/ql
